# Burnaby Rec availability monitor

1. install npm
2. in terminal, navigate to repo and npm install
3. npm start
4. open browser to http://localhost:3001/
5. (optional telegram notifications) Open a new chat with @BotFather on Telegram
6. Create a bot (/newbot) and check out its token with the /token command
7. Paste token access key into website, start the bot


When an availability is found, the signup link will automatically open.
Must leave terminal running to keep polling! Process will keep running if left on even if tab is closed. Kill terminal to stop process completely.
